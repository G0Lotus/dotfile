# HKKK's Dotfiles

## Install

- prerequisite
	- git
	- stow

Use `git clone` to clone this project into your `$HOME` directory
```shell
git clone https://github.com/G0Lotus/HkkkDotfiles $HOME
```
Run `stow` to symlink everything or just select what you want
```shell
stow */
```

```shell
stow <package>
```
